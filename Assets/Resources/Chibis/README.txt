Put chibi sprite sheets here.

Expected resource names for the single chibi:
- Shime_Walk
- Shime_Idle
- Shime_Lift
- Shime_Drop
- Shime_Hit

How it works:
- Walk frames play during idle movement.
- Lift frames play while dragging (held in air with wobble).
- Drop frames play while falling.
- Hit frames play for the floor impact moment.

If some set is missing, the actor still works and uses available frames.

Numbered PNG format is also supported (recommended if artist already exported frames):
- Shime_Walk_1, Shime_Walk_2, ...
- Shime_Idle_1, Shime_Idle_2, ...
- Shime_Lift_1, Shime_Lift_2, ...
- Shime_Drop_1, Shime_Drop_2, ...
- Shime_Hit_1, Shime_Hit_2, ...

Lower-case names also work:
- shime_walk_1, shime_walk_2, ...
